# deneme

Bu depo, LLM pozisyon kodlamasını görselleştiren tek dosyalık bir HTML demo içerir.

## İçerik

- [index.html](index.html): İnteraktif görselleştirici

## Nasıl kullanılır

Tarayıcıda [index.html](index.html) dosyasını açmanız yeterlidir.

## GitHub Pages

Bu proje GitHub Pages ile doğrudan yayınlanabilir. Repo kökünde [index.html](index.html) bulunduğu için ek bir build adımı gerekmez.

Yayınlamak için:

1. GitHub repo ayarlarına girin.
2. `Pages` bölümünü açın.
3. Kaynak olarak `Deploy from a branch` seçin.
4. Branch olarak `main` ve klasör olarak `/ (root)` seçin.

Yayınlandıktan sonra GitHub size paylaşabileceğiniz bir site linki verir.# deneme
